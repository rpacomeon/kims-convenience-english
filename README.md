# 김씨네 편의점 영어학습 앱 📚

> Kim's Convenience 드라마 자막 데이터를 활용한 30일 영어학습 챌린지

## 🎯 주요 기능

- **30일 커리큘럼**: 에피소드 13개를 4주에 체계적으로 분배
- **SM-2 간격반복**: 과학적 복습 시스템으로 효율적 암기
- **4종 퀴즈**: 한→영, 영→한, 빈칸 채우기, 문법 교정
- **8개 상황별 카테고리**: 인사, 쇼핑, 가족, 감정, 부탁, 직장, 일상, 조언
- **구동사 집중 학습**: 드라마 속 실제 예문과 함께
- **Mr. Kim의 비문법 학습**: 의도적 오류를 교육 기회로 활용
- **모바일 최적화**: 스마트폰에서 편하게 학습

## 📁 프로젝트 구조

```
김씨네편의점/
├── requirements.txt          # 의존성 (streamlit, pandas)
├── README.md                 # 이 파일
├── 김씨네 편의점.txt         # 자막 데이터 (5,693줄)
├── src/                      # 소스 코드
│   ├── app.py                # 메인 Streamlit 앱 (6개 페이지)
│   ├── data_loader.py        # 데이터 로드 및 파싱
│   ├── expression_extractor.py  # 핵심 표현 추출
│   ├── categorizer.py        # 상황별 분류
│   ├── vocabulary_builder.py # 어휘 분석
│   ├── broken_english.py     # 비문법 감지 및 교정
│   ├── quiz_engine.py        # 퀴즈 생성
│   ├── spaced_repetition.py  # SM-2 간격반복
│   ├── curriculum.py         # 30일 커리큘럼
│   └── ui_components.py      # UI 컴포넌트
├── config/                   # 설정 파일
│   ├── broken_patterns.json  # 비문법 패턴 (20개)
│   ├── categories.json       # 카테고리 정의 (8개)
│   └── phrasal_verbs.json    # 구동사 데이터 (25개)
└── learning_data.json        # 학습 진도 (자동 생성)
```

## 🚀 실행 방법

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

### 2. 앱 실행

```bash
cd src
streamlit run app.py
```

### 3. 브라우저 접속

- 로컬: http://localhost:8501
- 모바일 (같은 WiFi): http://<PC-IP>:8501

## 📱 6개 페이지

### 1. 오늘의 학습
- 현재 Day/Week 표시
- 오늘의 학습 초점
- 오늘의 구동사
- 새 표현 10~15개
- 복습 대기 항목

### 2. 에피소드별 학습
- 에피소드 선택 (1~13)
- 핵심 표현 20개
- Mr. Kim의 비문법 표현

### 3. 상황별 표현
- 8개 카테고리 그리드
- 카테고리별 표현 리스트

### 4. 구동사 마스터
- 드라마에서 사용된 구동사
- 실제 예문 3개씩
- 빈도 순위

### 5. 퀴즈
- 4종 퀴즈 유형 선택
- 즉시 피드백
- 오답 노트
- 정답률 통계

### 6. 학습 기록
- 30일 진행률
- 에피소드별 완료 현황
- 복습 통계
- 정답률 추이

## 🧠 학습 과학 원리

| 원리 | 적용 방식 |
|------|----------|
| **간격반복 (SM-2)** | 1일 → 3일 → 7일 → 16일 → 35일 복습 |
| **청킹 (Chunking)** | 단어가 아닌 전체 표현 단위 학습 |
| **능동적 회상** | 퀴즈로 직접 떠올리기 (수동 읽기보다 2배 효과) |
| **문맥 학습** | 드라마 장면/상황과 함께 제시 |
| **망각곡선 대응** | 복습 간격 자동 계산 |
