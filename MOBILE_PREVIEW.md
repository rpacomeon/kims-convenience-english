## 💡 사용자 경험 테스트

### 터치 인터랙션
1. 카드 탭 → 한국어 표시/숨김
2. 버튼 클릭 → 즉각 피드백
3. 스와이프 → 부드러운 스크롤
4. 선택지 선택 → 명확한 하이라이트

### 네비게이션 흐름
1. 오늘의 학습 → 표현 확인 → 퀴즈
2. 에피소드별 → 표현 학습 → 비문법 확인
3. 상황별 → 카테고리 선택 → 표현 리스트
4. 구동사 → 익스팬더 열기 → 예문 확인
5. 퀴즈 → 문제 풀기 → 결과 확인
6. 학습 기록 → 진행률 확인

### 오류 처리
- 데이터 로딩 실패 → 재시도 버튼
- 퀴즈 오답 → 설명 표시
- 네트워크 오류 → 안내 메시지

## 🎯 모바일 UX 베스트 프랙티스

1. **한 손 사용 가능**: 중요 버튼은 하단 또는 중앙
2. **즉각 피드백**: 모든 인터랙션에 시각적 피드백
3. **명확한 상태**: 현재 위치와 진행 상황 표시
4. **실수 방지**: 중요 액션은 확인 요청
5. **빠른 로딩**: 스켈레톤 UI 또는 프로그레스바
6. **오프라인 대응**: 캐시 활용 (향후 구현)
7. **접근성**: 색상 대비, 폰트 크기, 스크린 리더
8. **제스처**: 직관적인 스와이프/탭
